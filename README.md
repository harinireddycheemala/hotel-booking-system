# 🏨 Hotel Booking System

## 📌 Project Description
This project is a Hotel Booking Database System developed using SQL.
It manages hotel operations such as customers, rooms, bookings, and payments.

## 🚀 Key Feature
Stored Procedure for Room Availability Check:
- Takes check-in and check-out dates
- Prevents double booking
- Returns available rooms

## 🗂 Database Tables
- Customers
- Rooms
- Bookings
- Payments

## 🛠 Technologies Used
- MySQL
- SQL (Stored Procedures)

## ▶️ How to Use
1. Import `database.sql` into MySQL
2. Run:
   CALL CheckAvailability('2026-04-03', '2026-04-05');

## 📊 Output
- Available rooms
- Booking records
- Total revenue
